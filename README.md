<h1 align="center">Bootstrap Grid Layout</h1>

Layout responsive realizzato utilizzando la griglia di Bootstrap 5,
con adattamento del numero di colonne e della disposizione dei contenuti
in base alla larghezza del dispositivo.

## Obiettivo

- Riprodurre il layout fornito negli screenshot utilizzando la griglia di Bootstrap 5
- Adattare la disposizione delle colonne per:
  - desktop
  - tablet
  - mobile
- Sfruttare le utility class di Bootstrap per:
  - centralizzare il testo (es. `text-center`, `text-white`)
  - gestire spaziature e allineamenti (es. `p-*`, `m-*`)
  - utilizzare `row-cols-*` dove possibile per impostare il numero di colonne nelle righe
